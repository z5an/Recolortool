Recolor Tool

Dieses Tool ermöglicht es dir, Texturen in deinem Minecraft Texturepack mit einer benutzerdefinierten Farbe zu recolorieren. Es ist einfach zu bedienen und funktioniert mit PNG-Dateien. Bitte beachte die folgenden Schritte und Hinweise, um das Programm korrekt zu nutzen.

🚀 Anwendung

1. Vorbereitung:

Ordnerstruktur:

Stelle sicher, dass du das Projekt in einem Ordner hast, der die folgenden Dateien enthält:

recolor_tool.py (Das Python-Skript).

recolor_tool.bat (Batch-Datei zum Starten des Programms).

Ein Ordner namens input, in dem du die Texturen (PNG-Dateien) ablegst, die du recolorieren möchtest.

Ein Ordner namens output, in dem die bearbeiteten Texturen gespeichert werden.





2. Wie starte ich das Tool?

Öffne den Ordner, der das Tool enthält, und starte die recolor_tool.bat Datei.

Wichtig: Starte niemals die .py Datei direkt, sondern immer nur die .bat Datei. Die .bat Datei sorgt dafür, dass das Skript in der richtigen Umgebung ausgeführt wird.



3. Texturen ablegen:

Platziere deine PNG-Dateien im Ordner input.

Das Programm wird alle PNG-Dateien aus diesem Ordner verarbeiten.



4. Farbauswahl:

Nach dem Starten des Programms wird ein Fenster geöffnet, in dem du eine Farbe für die Texturen auswählen kannst. Wähle die gewünschte Farbe und bestätige deine Auswahl.



5. Ergebnisse:

Nachdem das Tool die Texturen verarbeitet hat, findest du die bearbeiteten Texturen im Ordner output.




🛠️ Voraussetzungen

Um das Tool erfolgreich auszuführen, sind die folgenden Programme und Module erforderlich:

1. Python Version:

Python 3.6 oder höher wird benötigt.

Falls du Python noch nicht installiert hast, kannst du es von der offiziellen Python-Website herunterladen.



2. Benötigte Python-Module:

Pillow (für die Bildbearbeitung)


Installiere es, indem du den folgenden Befehl in der Kommandozeile ausführst:

pip install Pillow
